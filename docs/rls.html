Análisis de Regresión con R

Chapter 2 Regresión lineal simple

Considere una muestra de tamaño \(n\), el modelo de regresión lineal simple para el \(i\)-ésimo individuo está dado por:

\[y_i=\beta_0+\beta_1 x_i + e_i,\ \ \text{para $i=1, \ldots, n$}\]

donde los supuestos del modelo son:

  • \(e_i\) tiene una distribución normal para cada \(i=1, \ldots, n\)

  • \(E(e_i)=0\) y \(Var(e_i)=\sigma^2\), de donde \(E(Y_i)=\beta_0+\beta_1X_i\) y \(Var(Y_i)=\sigma^2\)

  • Los \(e_1, e_2, \ldots, e_n\) son independientes.

Podemos agregar figuras directamente así:

y <- rnorm(100)
plot(density(y), lwd=2, col='blue3')
